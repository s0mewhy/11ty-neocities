my personal website; generates static pages with 11ty. requires node.js and npm. 

# getting started

1. clone the repo:

`git clone https://github.com/s0mewhy/11ty-neocities.git && cd 11ty-neocities`

2. install dependencies:

`npm install`

3. start the instance:

`npm start`
